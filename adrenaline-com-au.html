Using a promo code is easy and you can use the following steps to get amazing savings on your Adrenaline purchase :

Step 1: Pick your promo code

After you have picked the code that you’d like to redeem, you can click on “Get Code” to reveal the code.

Step 2: Retrieve the promo code

Highlight and copy the code from the box by right-clicking and selecting ‘Copy.’

Step 3: Get shopping!

The next step is where all the fun is…shopping. Choose the Adrenaline item/s you would like to buy and add to the cart/bag. Once you’re done browsing and ready to buy, move on to Step 4 for retrieving your discount.

Step 4: Input the code at the Adrenaline checkout stage

Look for the code box at the checkout stage that says “Promo Code” and simply paste your code in here. The image below is an example of where you paste the code in case you are facing any difficulty.

It is just as easy to find the promo code box if you are using a mobile phone as it is also usually visible at checkout.

Step 5: There you go!

Complete the checkout process by entering the payment and shipping details. We hope you had a great shopping experience!
